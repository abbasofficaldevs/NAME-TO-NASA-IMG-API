## 🌟 About Project

</div>

**NASA Name Image API** is a lightweight Flask API that turns any name into a custom image using NASA‑themed letter tiles.  
It randomly picks from three different variants for each letter (A‑Z) stored locally, combines them into a single image, uploads it to tmpfiles.org, and returns a direct URL – **zero API key required.**

Perfect for Telegram bots, Discord bots, profile picture generators, games, or any project that needs personalised name images.
