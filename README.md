## 🧠 What Is This?

</div>

**NASA Name Image API** is a no‑key, no‑nonsense image generator.  
Feed it a name (like `“Armstrong”`), and it returns a unique, horizontally stitched image where each letter is a different **NASA‑themed tile** – randomly chosen from three style variants per character.

No registration. No rate limits (within reason). Just JSON with a direct image URL.

## ⚡ Core Features

</div>

```diff
+ 🖼️  Name → Image in < 1 second
+ 🔀  Every generation is visually unique (random variant selection)
+ 📂  No database – works 100% from local letter tiles
+ ☁️  Auto‑uploads to tmpfiles.org – you get a permanent(‑ish) URL
+ 🔧  Easy to self‑host – just drop your letter images
+ 🧩  Ignores numbers & symbols – only A‑Z matters
+ 📦  Lightweight – runs on a Raspberry Pi or a $5 VPS
